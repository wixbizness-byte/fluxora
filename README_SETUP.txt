FLUXORA STUDIO — QUICK START

1. Extract this ZIP.
2. Open PowerShell inside the extracted folder.
3. Run:

   npm.cmd install
   npm.cmd run dev

4. Open http://localhost:3000

Production check:

   npm.cmd run build

The project is ready for GitHub and Vercel. All placeholder links currently
point to page sections or #. Replace them once the final products, prices,
community link, checkout links, testimonials and screenshots are confirmed.

Main files:
- app/site.tsx       Page structure and copy
- app/globals.css    Complete visual design and responsive styling
- app/layout.tsx     Browser title and metadata
